# 《廢星牧魂者》總覽

## 讀者一句話賣點
主角在「全星域垃圾終端」撿破爛，靠鍊成修復和靈魂刻印，量產鼠群從底層反殺到神座。

## 你現在要看的 5 份主檔
- `01-世界與規則.md`：世界觀、地圖分區、本地居民、生存規則。
- `02-能力與修煉.md`：金手指、資源循環、修煉階位、限制與平衡。
- `03-角色勢力與主線.md`：角色、勢力、第一卷主線與節奏。
- `04-撰寫規則.md`：每章字數、敘事節奏、Markdown 呈現風格與交稿自檢。
- `05-第一卷檢核與第二卷目標.md`：第一卷達成度檢核、未結清伏筆與第二卷目標骨架。

## 核心循環（爽感來源）
撿到寶物線索 -> 補齊缺件 -> 修復出高價值裝備 -> 擴編鼠群 -> 奪地盤 -> 拿到更高級垃圾投放權。

## 世界底層邏輯
- 這裡既是古文明神戰墳場，也是各星域最低成本垃圾投放地。
- 好東西很多，但危險更多；主角強在「看得懂 + 修得好 + 用得快」。
- 金手指不是無限製造，而是把廢料重組成可用資產。
