## 新鼠種詳細設定

### 獵殺鼠（集群剿滅單位）

**設計思路**：
*   **哨戒鼠 = 眼睛**：純粹的感知單位，脆弱但敏銳。
*   **噬靈鼠 = 獠牙 (特種)**：針對靈魂與能量護盾的干擾單位。
*   **獵殺鼠 = 獠牙 (物理)**：針對實體目標的集群毀滅單位。

噬靈鼠的定位是「精準干擾」——切斷單體意識鏈、防禦性部署。獵殺鼠則是純粹的「攻擊輸出」——以30-50隻為一個獵殺群，執行對建築物、據點、小型團體的清掃任務。

**核心特徵**：
- **體型**：比普通哨戒鼠大一圈，肌肉密度高，前齒經過鍊成爐強化（可咬穿標準制式護甲的關節縫隙）。
- **戰術模式**：不是單兵作戰，而是「群體包圍 → 切斷退路 → 多點同時攻擊」。由1隻哨戒鼠擔任指揮節點。
- **特殊能力**：「血霧共鳴」——獵殺群在戰鬥中釋放微量神性脈衝，互相連結為統一行動體。群體越大、殺傷越多，共鳴越強，攻擊協調性越高。
- **信仰輸出**：戰鬥中產生的信仰品質極高（「狂熱信仰」），但戰鬥結束後迅速衰減至中等。屬於爆發型信仰來源。

**平衡限制**：
- 獵殺鼠的刻印需要更高品質的神性材料（消耗噬靈鼠-β的兩倍）。
- 「血霧共鳴」持續時間有限（約15分鐘），超時後群體會陷入狂躁期，需要祈禱鼠現場安撫。
- 不適合長期防禦——它們是「進攻工具」，待命時信仰輸出低落且情緒不穩。

**建議出場時機**：Ch. 81-85，李塵從防守轉向反攻時首次部署。

### 煉骸鼠（戰場生物質回收轉化單位）

**設計思路**：戰場上最大的浪費是屍體。煉骸鼠將「死亡」轉化為「資源」——它們是移動的微型鍊成爐，專門處理有機物質。

**核心特徵**：
- **體型**：肥短，腹部膨脹，皮膚呈暗紅色（神性迴路外顯）。不具備戰鬥能力，需要其他鼠種保護。
- **轉化機制**：吞噬屍體或有機廢料後，體內的「微型煉化腔」將生物質分解為三種產物：
  1. **骸能核**（高價值）：拳指大小的暗紅色結晶，是濃縮的生命能量，可供種母鼠攝入或直接注入鍊成爐。
  2. **基礎營養膠**（中價值）：灰色膠狀物，可作為鼠群通用口糧，替代常規食物來源。
  3. **殘渣粉末**（低價值）：不可用廢料，排出體外。
- **轉化效率**：一隻煉骸鼠處理一具人類體量的屍體約需2小時，產出1枚骸能核 + 足夠50隻鼠3天的營養膠。
- **特殊效果**：處理攜帶神性殘留的屍體（如教團信徒）時，骸能核的品質會大幅提升——教團虹吸的生命質反而成為養分。

**平衡限制**：
- 轉化過程不可中斷，中斷會導致煉化腔過載，煉骸鼠當場死亡。
- 長期轉化被污染物質會導致煉骸鼠自身突變（污染累積值），需要定期由灰醫或祈禱鼠進行「淨化」。
- 骸能核有保質期（約72小時），過期後能量衰散，不可儲存。

**建議出場時機**：與獵殺鼠同步（Ch. 81-85），戰場清掃階段投入。

### 種母鼠（高信仰快速繁殖單位）

**設計思路**：傳統的鼠群擴編依賴母巢殘片的量產模組，速度穩定但上限明確。種母鼠是「生物活體量產」——攝入骸能核後直接在體內孕育預刻印幼鼠，跳過了鍊成爐的刻印排程。

**核心特徵**：
- **體型**：是普通鼠的三倍大，毛色銀白（高純度信仰迴路外化），動作遲緩但生命力極強。
- **繁殖機制**：攝入1枚骸能核後，進入「孕育週期」（約6小時），可產下8-12隻預刻印幼鼠。幼鼠出生即帶有基礎魂印，無需李塵逐一刻印。
- **幼鼠品質**：預刻印幼鼠的信仰品質取決於兩個因素——
  1. 骸能核的品質（來源越強的生物質 → 品質越高）。
  2. **祈禱鼠的臨場灌注**：如果繁殖時有祈禱鼠在場進行「信仰灌注」，幼鼠的初始信仰品質可從「本能」直接跳至「依賴」甚至「崇敬」。**這是祈禱鼠最重要的新角色。**
- **幼鼠分化**：預刻印幼鼠出生後為「未定型」，根據前72小時的環境和需求自動分化為最需要補充的鼠種。戰損多哨戒鼠 → 幼鼠傾向分化為哨戒鼠。
- **種母鼠上限**：同一時間最多維持3隻種母鼠（超過會互相干擾信仰迴路）。

**平衡限制**：
- 種母鼠孕育期間完全喪失行動力，必須被保護在安全區域。
- 每次繁殖消耗種母鼠自身10%的生命力。連續繁殖超過5輪需要至少72小時恢復期，否則種母鼠死亡。
- 預刻印幼鼠的品質上限永遠低於李塵親手刻印的精英個體——量大但精度不足。

**建議出場時機**：Ch. 83-85，戰爭進入消耗階段時，李塵被迫開發這條快速補兵路線。

## 戰爭循環（War Cycle）

第三卷中後期解鎖的核心戰術閉環：

```
                    ┌─────────────────────────────────┐
                    │                                 │
                    ▼                                 │
  ┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐
  │ 獵殺鼠群  │──→│ 煉骸鼠    │──→│ 種母鼠    │──→│ 新生幼鼠   │
  │ 集群剿滅  │   │ 屍體轉化  │   │ 快速繁殖   │   │ 補充戰損   │
  └──────────┘   └──────────┘   └──────────┘   └──────────┘
       ▲               │              ▲               │
       │               │              │               │
       │          骸能核/營養膠    祈禱鼠灌注      分化為各鼠種
       │               │              │               │
       └───────────────┴──────────────┴───────────────┘
```

**循環效率**：
- 一次成功的剿滅行動（30隻獵殺鼠 vs 約10名敵方人員）→ 約8-10枚骸能核 → 2隻種母鼠同時孕育 → 16-24隻新生幼鼠 → 72小時後補充回戰鬥序列。
- 如果祈禱鼠全程灌注，新生幼鼠的初始信仰品質可達「依賴」級，信仰回路壓力大幅降低。

**戰略意義**：
- **自給自足**：不再依賴外部神性材料或母巢殘片排程，戰場本身就是資源。
- **越戰越強**：每次戰鬥的戰損可以在72小時內用敵人的「屍體」補回，甚至超額補充。
- **恐怖威懾**：敵人發現「殺了他們的鼠，他們會從你的屍體裡長出更多」。

## 祈禱鼠的角色擴展

祈禱鼠在鼠群體系中的地位，隨著第三卷劇情推進發生質變：

| 階段 | 角色 | 說明 |
|---|---|---|
| 第一卷 | 信仰匯聚中轉 | 被動收集各鼠種的信仰，匯入主角靈魂海 |
| 第二卷 | 神格共振錨點 | 參與碎片融合，維持神火不滅 |
| 第三卷前期 | 意識護盾中繼 | 鍊成爐二轉後，祈禱鼠成為「多線程共鳴」的承載節點——每隻祈禱鼠可承載2人份的意識護盾 |
| 第三卷中後期 | **信仰灌注者** | 在種母鼠繁殖時進行「臨場灌注」，決定幼鼠的初始信仰品質。**沒有祈禱鼠，種母鼠只能產出「本能」級幼鼠** |
| 第三卷中後期 | **戰場牧師** | 隨獵殺群出擊，維持「血霧共鳴」的穩定性，防止獵殺鼠狂躁失控。一隻祈禱鼠可穩定一個30隻的獵殺群 |

**祈禱鼠成為了整個戰爭循環的「品質瓶頸」**——打仗靠獵殺鼠，回收靠煉骸鼠，繁殖靠種母鼠，但決定輸出品質的永遠是祈禱鼠。

## 戰爭循環的倫理陰影

**對「神性冷淡」主題的強化**：戰爭循環是一套極其高效的系統，但它的本質是：**用敵人的屍體養出更多士兵**。

- 蘇芮會質問：「你在把死人變成武器。這跟教團把活人變成燃料有什麼區別？」
- 李塵的回答（如果他還保有人性的話）：「教團吸的是活人。我用的是已經死了的敵人。」
- 蘇芮的反問：「但你開始**期待**他們死，不是嗎？每一個死的敵人，你的第一反應不是『又少了一個威脅』，而是『又多了一枚骸能核』。」

**這條線是「神性冷淡」從個人症狀升級為系統性危機的關鍵——當整個戰爭機器都依賴「死亡轉化」時，和平反而成為了資源匱乏。**

## 可持續爽點模板
- 小爽：獵殺群首戰告捷，30隻鼠在三分鐘內清掃一個教團哨站。
- 中爽：鼠群分工成功，低成本打贏高成本敵人。戰後煉骸鼠清場，種母鼠連夜繁殖，72小時後兵力反增。
- 大爽：祈禱鼠全程灌注，一批新生幼鼠信仰品質直接達到「崇敬」級，信仰值爆表。
- 隱爽：戰後清點信仰值，發現不知不覺又漲了一截。
- 暗爽：敵人發現殺了鼠群只會讓鼠群更多，開始恐懼。
