# 金手指：廢界鍊成爐

## 載體形態與融合過程

### 初始形態：古代神器殘核
- 外觀：拳頭大小、表面刻滿磨損銘文的金屬碎片，微弱發光。
- 來源：古代神戰遺物，是某位墜落神祇的「權柄工具殘片」。
- 主角在第1章從廢船裂縫深處撿到，碎片嵌入手掌傷口。

### 融合時間線
| 階段 | 章節 | 描述 |
|---|---|---|
| 外置期 | 第1章 | 碎片嵌入手掌傷口，微弱發光，無法取下 |
| 啟動期 | 第2章 | 碎片侵蝕手掌，高燒昏迷中與主角靈魂初步共鳴，鍊成爐意識首次現聲 |
| 過渡期 | 第3章 | 主角使用鍊成爐最低權限完成第一次修復 |
| 融合完成 | 第3章末 | 碎片耗盡外殼能量，核心鑽入主角靈魂海，外殼碎裂成廢鐵 |
| 內化運作 | 第4章起 | 金手指變為腦內介面，所有操作在意識中完成 |

### 融合代價
- 短暫失去觸覺（約一天恢復）。
- 靈魂海出現第一道「裂隙」，後續成為容納神格碎片的空間，也是風險源。

## 鍊成爐核心能力
- **分解**：將廢料拆成基礎元素。
- **提純**：分離雜質，提升材料品階。
- **修復**：對照「缺件清單」重組損壞物品。
- **重組**：將多種材料合成新物件。
- 重點優勢：可顯示目標物「缺件清單」與最近補件來源。

## 資源循環（主角成長引擎）
1. 掃描垃圾 -> 判定價值。
2. 分解低值廢料 -> 拿基礎材料。
3. 補齊缺件 -> 修復高值設備。
4. 把利潤投入工坊與鼠群擴編。
5. 用鼠群搶更高級投放區與神性材料。

## 平衡限制（避免一瞬爆棚）
1. 不能無中生有，只能重組既有材料。
2. 鍊成負荷值每日上限固定，超載會重傷。
3. 產線槽位有限，擴產要升級工坊。
4. 材料品質決定成品上限，不能越級白嫖。

## 與信仰體系的關聯
- 鍊成爐的能量上限會隨「信仰之力」的積累而緩慢提升。
- 鼠群的信仰回饋 = 鍊成爐的隱性充能來源。
- 這讓「牧鼠」和「鍊成」兩條線互相支撐，缺一不可。
