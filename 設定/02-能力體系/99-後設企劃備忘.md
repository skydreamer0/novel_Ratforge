# 後設企劃備忘（非世界觀設定）

## 資源循環（主角成長引擎）
1. 掃描垃圾 -> 判定價值。
2. 分解低值廢料 -> 拿基礎材料。
3. 補齊缺件 -> 修復高值設備。
4. 把利潤投入工坊與鼠群擴編。
5. 用鼠群搶更高級投放區與神性材料。

## 平衡限制設計理由
1. **不能無中生有**：限制鍊成爐只能重組既有材料，防止主角憑空創造資源。
2. **鍊成負荷值**：每日上限固定，防止主角無限使用能力，增加策略性。
3. **產線槽位限制**：擴產需升級工坊，模擬經營類的升級感。
4. **材料品質決定成品上限**：防止主角低成本製造神器，強調資源獲取的重要性。
5. **神格排異**：融合碎片有失敗率，作為風險控制機制。
6. **信仰衰減**：強制主角持續維護鼠群連結，防止「種田」變成單純數值堆疊。

## 可持續爽點模板

### 鍊成爐爽點
- **小爽**：補齊缺件，立刻修好值錢裝備，獲得即時收益。
- **中爽**：解鎖新配方或功能模組，提升生產效率或戰鬥輔助能力。
- **大爽**：完成階段性升級（如二轉、三轉），解鎖全新的規則級能力（如全域廣播）。

### 修煉爽點
- **小爽**：新刻印一批鼠群，信仰值跳升，鍊成爐能量上限突破。
- **中爽**：鼠群從本能信仰升級為依賴信仰，戰力質變。
- **大爽**：信仰積累觸頂 + 融合神格碎片 + 當場突破新階。
