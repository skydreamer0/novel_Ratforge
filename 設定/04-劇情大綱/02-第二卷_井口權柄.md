# 第二卷：井口權柄

## 卷核心一句話
主角帶著第一枚神格憑證進入黑潮牌桌，爭奪零號墜神井井口權柄，從守塔者升級為星域供應鏈玩家。

## 第二卷五大目標（不變）
1. 取得零號墜神井「分層貨權」至少一層，打開外域資源入口。
2. 建立T-17到灰環停泊場的安全運輸鏈，不再被單方斷供。
3. 找到修補魂譜裂痕與母巢活性的方法，避免主角戰力短命化。
4. 逼出黑潮與聖骸的真實上下游關係，拆掉其中一條控制鏈。
5. 拿到第二枚神格碎片線索，形成「1/3 -> 2/3」的卷末拉升。

## 卷末節奏骨架（第51-60章收尾）
1. 第51-53章：聯航獨立承壓與鬼影回溯。主角強制休整，聯航需證明體系價值。解決內鬼問題，為下井掃清障礙。
2. 第54-56章：雙鎖對沖與調虎離山。利用「群體信仰」破解井口概念鎖，用計調離黑潮主力。
3. 第57-60章：奪神之戰與井口權柄。二次進井，意識光譜對決，剝離第二枚神格碎片，確立星域新霸權。

## 升級與衝突同步規則
- 戰場升級：單塔守城 -> 航線路由 -> 井口配額與封條雙鎖。
- 反派升級：地方審判線 -> 特管清洗線 -> 黑潮高層與教團合流。
- 爽點升級：修復撿漏 -> 奪航權 -> 改規則 -> 爭井口分層貨權。
- 代價升級：裂痕惡化、封約反噬、供應鏈成本、聯盟票數風險同步上升。

## 第二卷寫作守則補充（後半）
- 保持「每3章一小高潮、每10章一大高潮」節奏，不改。
- 權謀章不可空談，必須落到可量化收益：貨權%、運價、供應量、窗口時長至少一項。
- 每章至少推進兩條線：行動線（打/潛/運）與規則線（驗簽/條款/證據）。
- 主角每次大勝後必帶一項可見代價，且代價需影響下一章決策。
