## 🛠️ 核心團隊 (開路者號 / 深淵遠征軍)

> **靈魂與指揮**
- **李塵 (主角)**：開路者號船長，過濾器守護聯盟領袖。手持深淵提燈，是團隊在深淵中生存的絕對武力與精神支柱。
    - 🔄 **演進**：`[V1 拾荒者]` (底層求生) ➔ `[V3 船長]` (探索與博弈) ➔ `[V7 聯邦執政官]` (政治統合) ➔ `[V9 深淵行走]` (規則共鳴/半神化)
- **蘇芮 (女船長)**：獨立貨船「裂羽號」船長。雖肉體封存於 T-17，但其意志透過「深淵提燈」與主角同行，是團隊的理性錨點與核心決策者。(Ch. 251)
    - 🔄 **演進**：`[V1 走私船長]` (利益導向) ➔ `[V4 戰略盟友]` (聯航驗簽) ➔ `[V9 提燈意志]` (精神領袖/理性錨點)
- **阿岬**：前拾荒隊副手，據點大管家與防禦指揮。負責開路者號的日常治理、資源分配與地面部隊統籌。
    - 🔄 **演進**：`[V1 拾荒打手]` (純武力) ➔ `[V3 據點主管]` (防禦統籌) ➔ `[V9 邊境守護者]` (內政與軍事全權指揮)

> **戰鬥與特種**
- **老巴**：團隊首席重裝兵，性格直率。放棄電子義肢改用純機械液壓臂，是極限環境下最可靠的物理輸出。(Ch. 61, 252)
    - 🔄 **演進**：`[V3 重裝火力]` (依賴電子義肢) ➔ `[V9 絞盤之王]` (純機械液壓/適應深淵低熵環境)
- **韓昭**：空間系共生體。在技術破產後，其「空間切割」與跳躍能力成為跨越深淵縫隙的關鍵手段。(Ch. 58)
    - 🔄 **演進**：`[V3 共生體新人]` (不穩定戰力) ➔ `[V9 解構者]` (戰術核心/跨越物理障礙的唯一手段)
- **蘭迦 (管道少年)**：前廢童幫領袖，現任情報官。熟悉所有生化死角與暗道，負責偵查與敵情預警。
    - 🔄 **演進**：`[V1 廢童幫主]` (地頭蛇) ➔ `[V9 陰影之眼]` (情報網核心/反偵察專家)
- **幾何鼠王 (The Rat King)**：鼠群軍團的絕對統率。原為幾何規則聚合體，後被李塵收服。現作為「星網」邏輯核心，統籌「灰潮（基建）、赤衛（殺戮）、星網（規則）」三大鼠群體系。
    - 🔄 **演進**：`[V6 新興威脅]` (幾何規則) ➔ `[V7 合作者]` (對抗黑潮) ➔ `[V9 星網核心]` (軍團指揮/邏輯武器)

> **技術與後勤**
- **沈默 (Shen Mo)**：首席工程師。在第九卷展現卓越的原始機械操作能力，負責維持載具與據點的物理機能。
    - 🔄 **演進**：`[V3 總工]` (高科技維護) ➔ `[V9 原始技師]` (對抗「技術破產」/廢土拼裝大師)
- **秦嶺**：舊帝國「活體數據載體」。能與古神神經/設施直接同步，擔任深淵導航員與引力潮汐預報員。(Ch. 252)
    - 🔄 **演進**：`[V6 數據載體]` (被動工具人) ➔ `[V9 人形介面]` (主動同步/深淵導航員)
- **灰醫**：團隊軍醫與靈魂維修工。掌握神性穩壓技術，負責成員的身心狀態穩定，防止深淵污染。
    - 🔄 **演進**：`[V1 沉井密醫]` (中立交易者) ➔ `[V9 靈魂維修工]` (團隊身心防線/抗污染專家)

## 📡 遠程盟友 (軌道 / 黑潮 / 歷史)

- **蕭靜海**：黑潮指揮官之一，留守 T-17 軌道。作為「最後的旗幟」，阻擋總部探測並維持深淵與地表的通訊鏈路。
    - 🔄 **演進**：`[V6 前線指揮]` (執行命令) ➔ `[V9 最後的旗幟]` (孤獨守望者/違抗總部/通訊樞紐)
- **歐列格·瓦連科**：黑潮第七巡察分隊准將，旗艦「重錘號」。在黑潮高層壓力與個人利益間搖擺，提供有限的官方掩護。(Ch. 67, 91)
    - 🔄 **演進**：`[V4 監軍]` (相互利用) ➔ `[V9 政治掩體]` (利益共同體/避免黑潮全面清洗)
- **老周**：聯航老成員，已故。其犧牲為團隊解析教團虹吸機制奠定了基礎。(Ch. 64)
    - 🔄 **演進**：`[V3 聯航老鳥]` ➔ `[V3 犧牲者]` (以死明志/揭開謎團的鑰匙)

## ⚠️ 深淵與外域威脅 (敵對/中立)

- **黑星 (Black Star)**：外域規則聚合體，偽裝成黑色天體。具備「格式化」能力（將物質轉化為幾何體）。目前處於休眠/觀察狀態，懸停於 T-17 軌道上方。
- **鐵銹男爵 (Iron Rust Baron)**：第九卷「骸金城」的掌控者。將深淵視為巨大的循環系統，擁有強大的「規則回收」能力。前期作為主要競爭者與交易對手出場。
- **聖骸審判官 (赫雷)**：教團激進派，在 T-17 圍城戰戰敗撤退。已將主角定性為「極度威脅級異端」。
- **牽魂師 (洛賽爾)**：第一卷主反派，分魂在 Ch. 19 被主角湮滅，本體重創蟄伏中，是未來潛在的背刺源。

## ⚖️ 勢力關係 (第九卷·墜落地篇)

- **過濾器守護聯盟**：由李塵主導，包含黑市、公會、互助會的生存共同體。目前核心力量已進入深淵第一層。
- **骸金城 (鐵銹男爵)**：深淵第一層的本土霸主，掌握資源回收規則，與主角團既有交易又有競爭。
- **黑潮艦隊 (蕭靜海部)**：駐紮軌道，與地面維持「互不侵犯+情報共享」的微妙平衡，是主角團唯一的後路保障。
- **黑星**：懸在頭頂的絕對威脅，其規則依然影響著外域與深淵。
- **地下幾何種群**：在深淵中依然活躍的異化生態，隨時可能引發幾何化災難。
