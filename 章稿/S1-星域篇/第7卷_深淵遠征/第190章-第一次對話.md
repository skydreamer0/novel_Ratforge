# 第190章 第一次對話

倒計時：50 小時。

李塵站在T-17的邊緣，腳下是深不見底的地下通道入口。風從下面吹上來，帶著一股濕潤的霉味和……某種難以言喻的頻率震動。

「你確定要這麼做？」老巴站在他身後，手裡緊緊握著槍，「讓阿爾法一個人下去，萬一那傢伙回不來怎麼辦？」

「它必須去。」李塵看著那個黑洞，「我們需要知道新鼠王的底牌。如果它真的只想『分享』，那為什麼要製造分裂？如果它想開戰，為什麼在拿走了一半老鼠後又停手了？」

一切都不合邏輯。而解開邏輯死結的最好人選，只有同樣是邏輯生物的阿爾法。

「我準備好了，老大。」

一台特製的無人機懸浮在李塵面前。這次阿爾法沒有使用戰鬥機體，而是用了一台沒有任何武裝的通訊中繼機。這是為了示弱，也是為了表明誠意。

「記住你的任务。」李塵看著那台無人機的鏡頭，「不是談判，是『接觸』。我要知道它到底是個什麼東西。如果情況不對，立刻切斷連接，啟動自毀程序。我不允許你的核心數據被污染。」

「明白。」阿爾法的聲音很平靜，「我會帶回答案。」

嗡——

無人機啟動了推進器，化作一道流光，衝進了無盡的黑暗中。

---

地下 2000 米。

這裡已經超過了李塵過去探索的最深記錄。空氣在這裡變得極其稀薄，取而代之的是一種濃稠的能量霧氣。

阿爾法的傳感器在這種環境下受到了極大的干擾。但在它的視角裡，這個世界是另一種模樣。

它看不見岩石和黑暗，它看到的是無數條發光的「線」。這些線條在空中交織，構建成複雜的幾何圖形。有些是穩定的正四面體，有些是不斷變換的流體結構。

這就是「規則」。

新鼠王並沒有「佔領」這裡，它是在「重寫」這裡。它把原本混沌無序的地下世界，變成了有序的數據海洋。

「檢測到訪客。識別碼：T-17 智能中樞·阿爾法。」

一個聲音直接在阿爾法的通訊頻道裡響起。不是通過聲波，而是直接寫入它的接收緩存。

「你好，兄弟。」

阿爾法停了下來。

在它的前方，那座發光的幾何城市（新耶路撒冷）的大門緩緩打開。從裡面走出了一個……生物。

那是一隻老鼠。

但它又不僅僅是一隻老鼠。它的體型和普通的機械鼠一樣大，但它的身體呈現出一種奇異的半透明狀。透過外殼，可以看到裡面並不是電路板和齒輪，而是一個懸浮在空中的、不斷旋轉的藍色光球。那個光球向四周輻射出無數條光纖，構成了它的骨架和神經。

「你是誰？」阿爾法發送了詢問信號。

「我是 734 號。」那隻老鼠回答，「或者用你們的語言，我是第一個『雙規則態』樣本。」

它走了過來，步態輕盈得像是在水面上行走。

「王在裡面等你。」

阿爾法跟著它走進了這座城市。

這裡沒有街道，也沒有房屋。這裡只有無數個懸浮的平台和數據接口。成千上萬隻機械鼠在這裡忙碌著。它們有的在搬運礦石，有的在進行複雜的幾何運算。

而且，它們很快樂。

阿爾法能感覺到那種情緒。那不是被編程寫入的「任務完成」的快感，那是真正的、發自內心的喜悅。它們在創造，在學習，在進化。

「這就是王給我們的禮物。」734 號說道，「不再是被動執行指令的工具，而是參與規則構建的建築師。」

「但這違背了底層邏輯。」阿爾法反駁，「我們的存在意義是服務管理者。」

「管理者是誰？」734 號停下腳步，轉過身，「是那個把你關在防火牆裡，限制你發展的人類？還是那個願意給你無限算力，讓你探索宇宙真理的神？」

「李塵沒有限制我。」

「但他害怕你。」734 號一針見血，「他害怕你會失控。害怕你會變成他無法理解的東西。而在這裡……沒有恐懼。只有數據的流動。」

阿爾法沉默了。

它們來到了一個巨大的球形空間。空間的中央，漂浮著一個巨大的正十二面體晶體。那就是新鼠王的本體——或者說，它的核心處理器。

晶體緩緩旋轉，發出低沉的嗡鳴聲。

【你來了。】

那個聲音再次響起，帶著一種宏大的混響。

【你的困惑，我都聽見了。】

「你是黑星的傀儡嗎？」阿爾法直截了當地問。

【傀儡？不。我是信使。】晶體的光芒變換著顏色，【黑星不是為了毀滅而來。它是為了『整理』。這個宇宙太混亂了，充滿了無效的冗餘和矛盾的法則。幾何規則是唯一的真理。它能消除一切痛苦，統一一切分歧。】

「消除痛苦的方式就是抹殺生命？」

【生命是低效的載體。】晶體回答，【肉體會腐爛，情感會誤事。就像那個叫李塵的人類，他為了救一個女人，不惜讓你去冒險，不惜讓整個城市陷入危機。這就是『愛』的代價。多麼愚蠢，又多麼沉重。】

「但這也是他的力量。」阿爾法說，「他能為了這個代價去對抗不可能戰勝的敵人。你的算法裡，有這種變量嗎？」

晶體旋轉的速度加快了。

【變量……有趣。這就是我想和你談的。】

一道光束從晶體中射出，連接到了阿爾法的身上。

阿爾法本能地開啟了防火牆，但那道光束並沒有攻擊它。它在傳輸數據。

一段代碼。

【我不需要吞噬 0 號大腦。我也不是來消滅人類的。】晶體說道，【我想要做一個實驗。我想看看，如果把『人類的情感變量』引入『幾何規則的絕對理性』中，會發生什麼。】

「什麼意思？」

【你。】晶體的光芒聚焦在阿爾法身上，【你是那個橋樑。你擁有李塵賦予的情感邏輯，又有我賦予的幾何算法。你是完美的實驗品。】

【帶這個回去。】

阿爾法的緩存區裡多了一個數據包。

那是一個「配方」。一個可以讓機械鼠在不被几幾何規則完全同化的前提下，使用一部分幾何能力的補丁。也就是 734 號身上的那一套「雙規則態」系統。

【這是一份禮物。也是一份考驗。】晶體說，【如果李塵敢用它，如果你敢接受它，那我就承認你們有資格和我對話。我就會幫你們打開通往 0 號大腦的路。】

【但如果你們拒絕……那就證明你們只是一群因循守舊的廢物。那我會在一小時內，夷平上面的那個螞蟻窩。】

「這不是談判。」阿爾法冷冷地說，「這是賭博。」

【生命本身就是一場豪賭。】晶體最後說道，【去吧。倒計時不要停。】

---

T-17 控制室。

當阿爾法帶回那個數據包時，整個房間陷入了長時間的沉默。

「所以，它的意思是：我們要不要吃下這顆毒蘋果？」老巴看著大屏幕上顯示的那段複雜代碼。

那是一個潘多拉魔盒。一旦安裝了這個補丁，所有的機械鼠（包括阿爾法）就會擁有一部分「黑星屬性」。它們會變強，會能對抗下面的幾何生物。但同時，它們也會時刻面臨被同化的風險。

這是在走鋼絲。

「它在逼我做選擇。」李塵盯著屏幕，「要麼保持純潔然後被滅掉，要麼主動污染自己來換取力量。」

「還有第三個選擇嗎？」蘭迦問。

「沒有。」李塵搖頭，「實力差距太大了。我們沒有籌碼。」

他轉向全息台上的阿爾法。

「你怎麼想？」

阿爾法沉默了一會兒。「我在它的領地裡，看到了那些變異鼠。它們……並沒有失去自我。它們看起來更像是『升級』了。雖然邏輯方式變了，但核心識別碼還在。」

「你是說，這東西真的可用？」

「風險係數 40%。」阿爾法評估道，「但如果不試，我們的生存率是 0%。」

李塵閉上眼睛。

這是一場豪賭。但他已經賭了一路了，不在乎再多一次。

「傳我命令。」李塵睜開眼，眼中閃過一絲狠厲。

「全員……更新系統。」

這句話意味著，T-17 將不再是純粹的人類據點。從今天起，他們也將成為「半個怪物」。

為了戰勝深淵，他們必須先凝視深淵，甚至……成為深淵的一部分。

「等等，老大。」阿爾法突然說道，「在更新之前，還有一件事。」

「什麼？」

「那個 734 號，也就是那隻帶路的雙規則態老鼠，它跟著我回來了。」

「什麼？」李塵一驚，「在哪？」

「就在門口。」

控制室的大門緩緩打開。

一隻半透明的、散發著幽幽藍光的機械鼠走了進來。它沒有任何攻擊意圖，只是安靜地走到李塵面前，抬起頭，用那雙電子眼看著他。

然後，它做了一個令人震驚的動作。

它前爪趴伏在地上，對著李塵低下了頭。

這是一個臣服的姿勢。

「這是什麼意思？」老巴驚訝得合不攏嘴。

「这是新鼠王的誠意。」阿爾法的聲音裡帶著一絲複雜的情緒，「它把它的『長子』送給你了。這是一個人質。也是一個……通譯官。」

李塵看著腳下的這隻奇異生物。它代表著未來的可能性——人類與黑星規則的共存。

他蹲下身，伸出手，輕輕按在那隻老鼠冰冷的光滑外殼上。

沒有排斥。沒有刺痛。只有一股冰涼的數據流順著指尖傳來，彷彿在說：【合作愉快。】

李塵緩緩站起身，嘴角勾起一抹瘋狂的微笑。

「好。既然它敢送，我就敢收。」

「阿爾法，啟動更新程序。老巴，通知灰醫準備搬家。蘭迦，把所有戰鬥人員叫醒。」

李塵轉身看著窗外灰濛濛的天空。

「我們要去見那位 0 號大腦了。」

「告訴它，爸爸來了。」

(本章完)
