# 第87章 聖像之心

灰環的地下水道系統就像是這座城市的腸道，扭曲、骯髒，而且充滿了未消化的殘渣。

李塵行走在齊膝深的污水中。防護服的過濾器發出不堪重負的嗡鳴聲，試圖將空氣中那種腐爛的沼氣過濾掉。在他身後，三十隻獵殺鼠無聲地潛游，紅色的眼睛在黑暗中閃爍，像是一串鬼火。

這條路是他通過解析之眼找到的。聖像的能量根鬚深深扎入地下，與城市的舊動力管網糾纏在一起。那一束最強、最純粹的金色能量流，就是導航標。

「距離目標還有三百米。」

通訊器裡傳來阿岬的聲音，伴著電流的雜音。「地面上的佯攻已經開始了。蘇芮帶著人在正門放煙火，聖裁騎士團被吸引過去了。但他們撐不了太久，那可是兩百個瘋子。」

「知道了。」李塵關掉通訊。

越靠近中心，那種神性的壓迫感就越強。空氣中不再是污水的臭味，而是一種令人作嘔的甜膩香氣——那是高濃度聖水揮發的味道。

這裡已經不是下水道了。

周圍的管壁上開始出現金色的結晶。那些原本生鏽的鐵管、發霉的磚塊，都被一層金色的類似苔蘚的物質覆蓋。那是神性輻射長期侵蝕的結果。

老鼠們明顯感到了不安。它們發出低沉的嘶鳴，爪子不安地抓撓著牆壁。但在「依賴級」信仰的強制命令下，它們依然緊緊跟隨，雖然每一隻都在顫抖。

「到了。」

李塵停下腳步。

在他面前，是一面巨大的金色屏障。那是聖像的地下基座防護層。它像是一面半透明的牆，阻擋了一切污穢的進入。

「解析之眼。」

視野切換。現實的顏色褪去，取而代之的是能量的流動。金色的屏障在他的眼中被拆解成無數條流動的能量線，它們交織、纏繞，構成了一個完美的防禦矩陣。

「結構強度：極高。物理破壞不可行。」
「弱點偵測：頻率共振點。每0.7秒出現一次相位偏移。」

李塵伸出手，按在屏障上。手套表面發出滋滋的燒蝕聲。

「影鼠，高頻噪音。」

十隻一直潛伏在獵殺鼠背上的影鼠跳了下來。它們張開嘴，喉嚨處的聲帶劇烈震動，發出了一種人耳聽不見，但在能量層面上極其尖銳的嘯叫。

聲波撞擊屏障，激起一圈圈漣漪。

滋滋滋——

金色的屏障開始顫抖，表面出現了水波一樣的紋路。那是結構不穩定的徵兆。

「獵殺鼠，咬合。」

三十隻獵殺鼠同時撲上去，對著那些不穩定的節點狠狠咬下。

「咔嚓。」

屏障碎了。不是被打破的，而是像一塊老化的玻璃一樣，因為頻率失調而崩解。金色的碎片落在污水中，瞬間熄滅。

李塵跨過碎屑，走進了聖像的核心區。

這是一個巨大的球形空間，直徑至少有五十米。四周的牆壁上刻滿了發光的符文。而在空間的中央，懸浮著一顆心臟。

不，那是一塊巨大的、呈心臟形狀的金色晶體，足有兩層樓那麼高。它在緩慢地搏動，每一次搏動，都會發出咚的一聲巨響，向地面輸送海量的神性波紋，維持著上面那座雕像的光輝。

「這就是……聖像之心。」

李塵走近了幾步，感覺呼吸都變得困難。這裡的神性濃度太高了，如果不開著意識護盾，他的靈魂早就被同化了。

鍊成爐的數據在瘋狂刷屏。

「發現高純度靈魂結晶。」
「預估神性含量：4000單位。」
「警告：檢測到上行數據鏈路。」

李塵愣了一下。

「上行鏈路？」

他仔細觀察那顆心臟。在那金色的光輝深處，他看到了一絲黑色的線。那不是雜質，那是數據線。這顆心臟不僅僅是在輸出能量，它還在……上傳？

上傳什麼？

李塵閉上眼，解析之眼全力運轉，試圖讀取那條黑色線路的信息。

一瞬間，無數嘈雜的聲音衝進了他的腦海。

「救救我……」
「我不舒服……」
「讚美聖子……」
「好痛，為什麼會這麼痛……」
「我不想死……」

那是灰環四萬居民的聲音。每一個接受過「聖觸」，每一個在家裡供奉小神像的人，他們的淺層意識都被這顆心臟連接了。

它像是一個巨大的吸塵器，在收集恐懼。收集痛苦。收集絕望。

然後，把這些負面情緒打包，通過那條黑色的線，發送到了遙遠的北方——在那裡，有一個更巨大的接收端。

「原來如此。」

李塵睜開眼，嘴角露出一絲嘲諷的笑。

「根本不是什麼賜福。這是一座養殖場。這座聖像就是個巨大的吸管，插在灰環所有人的腦袋上，吸食他們的靈魂。」

所謂的「安寧」，不過是被吸乾了反抗意志後的麻木。

「怪不得歐列格說教團只要靈魂。」李塵喃喃自語。「因為這才是最值錢的礦產。恐懼比黃金更值錢。」

現在，他要砸了這座礦。

「全體獵殺鼠。」

李塵在意識中下令。

「目標：上方連接管。咬斷它。」

三十隻獵殺鼠像紅色的閃電一樣竄了上去，爬滿了那根連接心臟和地面的粗大管道。那根管道裡流動著高濃度的聖水，是整個系統的冷卻液。

「吱！！！」

瘋狂的撕咬。

管道是用高強度合金做的，但在獵殺鼠那專門破甲的牙齒下，很快就變得千瘡百孔。

「嘶——」

金色的聖水從破口處噴湧而出，淋在老鼠們身上。

那是在燃燒。聖水對黑暗生物是劇毒。

這一次，獵殺鼠們沒有進化，而是在慘叫中融化。它們的皮毛脫落，肌肉溶解，露出了白森森的骨頭。空氣中瀰漫著皮肉焦爛的惡臭。

但沒有一隻老鼠鬆口。它們在死亡前，把最后一點力氣都用在了咬合上。哪怕下顎骨都爛掉了，它們依然死死卡在管道上。

「咔——崩！！！」

一聲巨響。

連接管斷了。

失去了上方的支撐，那顆巨大的金色心臟失去了平衡，重重地砸在了地上。

轟隆隆——

整個地下空間都在震動。管線斷裂，火花四濺。

與此同時，地面上。

正在和蘇芮對峙的聖裁騎士們突然停下了動作。

他們身後的聖恩像，那座屹立在灰環中心、象徵著教團絕對權威的金色巨人，突然黯淡了下來。

光，熄滅了。

緊接著，巨大的雕像表面出現了無數道裂痕，從基座一直蔓延到頭頂。

「不……」騎士隊長絕望地跪了下來，雙手顫抖著伸向天空。「神啊……」

「轟隆！！！」

高達百米的聖像在眾目睽睽之下崩塌。金色的碎片像雨點一樣落下，砸毀了廣場周圍的建築，揚起了漫天的塵埃。

但更可怕的事情在後面。

隨著聖像的毀滅，那種籠罩在灰環上空的「安寧力場」瞬間消失。

四萬名早已對這種力場產生生理依賴的居民，在同一時間崩潰了。

家屬區、貧民窟、商業街……無數人倒在地上，抱著頭尖叫、嘔吐、抽搐。他們的眼睛翻白，口吐白沫。那是靈魂層面的戒斷反應。

就像是一群癮君子被突然切斷了毒品。

整個城市都在哀嚎。

地下核心區。

李塵看著那顆摔碎的心臟。

它已經不再發光了，變成了一堆灰白色的粉末。而在粉末中，無數金色的光點正在逸散。

那是無主的純粹神性。

「鍊成爐。」李塵伸出手。「啟動吸收模式。」

「目標鎖定：游離神性。」
「開始捕獲。」

一個看不見的漩渦在李塵掌心成型。那些金色的光點被強行吸了過來，湧入他的體內。

暖流。

久違的暖流。

李塵感覺到靈魂深處的那道裂痕——因為二轉透支而產生的18%裂痕——正在被迅速填補。那種酥麻的感覺讓他忍不住想要呻吟。

18%……20%……25%……30%。

裂痕修復完畢。甚至還溢出了一些，開始強化他的靈魂韌性。

「吸收完成。」
「當前魂譜完整度：100%。」
「高強度輸出次數：重置為7/7。」

李塵握緊了拳頭。

他感到前所未有的強大。那種虛弱感消失了，取而代之的是一種充盈的力量。

看著地上那些為了咬斷管道而溶化成血水的獵殺鼠屍體，李塵的眼神變冷。

「你們的犧牲是有價值的。」他說。

他轉身離開。身後，那是已經徹底死去的聖像之心。

當他走出下水道，回到地面的時候，正好看到聖像崩塌的塵埃尚未落定。

月光下，廢墟之上。

阿岬、韓昭、蘇芮，還有無數T-17的戰鬥人員，正呆呆地看著那座曾經不可一世的神像變成一堆瓦礫。

他們看到了李塵。

他渾身濕透，沾滿了淤泥和血水，手裡還提著半截斷裂的金色管道。

那一刻，沒有欢呼。

只有敬畏。

因為他們知道，從這一刻起，李塵不再是一個塔主。

他是弒神者。

「斬首行動」的結算報告在塵埃瀰漫的空氣中緩緩浮現。目標：灰環聖恩像——已摧毀。教團的靈魂採集鏈路被徹底切斷，整座城市陷入了戒斷反應的混亂。代價是「甲群」獵殺鼠的全軍覆沒，它們化作了那一灘灘血水。但回報是絕對的：吸收的聖像碎片徹底修復了他靈魂譜系的裂痕，將其推升至完美的30%完整度。系統的最終評價只有四個字：【神話破滅】。他親手將高高在上的教團從雲端拉進了泥坑。
