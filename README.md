# 廢星牧魂者

**English Title:** Ratforge: Soul Shepherd of the Waste Star  
**Short File Name:** `Ratforge`

## 一句話簡介
主角墜入全星域垃圾場，靠廢料鍊成與魂譜刻印量產鼠群，從底層拾荒一路打進神格與井口權柄之爭。

## 故事介紹
這裡是古文明神戰墳場，也是各星域最低成本的垃圾投放地。  
在鐵屑環帶、鼠籠城、拋投塔群與零號墜神井之間，主角李塵以「看得懂廢料、修得出價值、用得快」的能力建立生存優勢。  
他先以鍊成爐修復破爛裝備換取資源，再透過魂譜刻印建立鼠群信仰回路，逐步擴張據點、反殺獵手、爭奪航道與配額，最終走向神格進階與星域規則改寫。

## 核心看點
- 廢料鍊成：撿破爛 -> 補缺件 -> 修高價裝備 -> 擴產線。
- 鼠群牧魂：從單體刻印到模板量產，建立可持續信仰回路。
- 生存權謀：黑市、公會、教團、物流盟多方博弈，從打架升級到制度戰。
- 神格主線：神格碎片不是外掛，而是信仰與代價累積後的催化突破。

## 網頁閱讀器
本專案包含一個輕量級的網頁閱讀器，方便預覽與閱讀章節內容。
- **入口**：直接瀏覽器打開 `index.html`。
- **功能**：
  - 自動掃描並列出 `章稿/` 下的所有章節。
  - 支援 Markdown 渲染與樣式優化。
  - 具備深色模式、章節搜尋與目錄導航功能。

## 專案結構
- `章稿/`：各章正文草稿（目前更新至第五卷）。
- `設定/`：世界觀、能力體系、角色檔案、劇情大綱與寫作規則。
- `index.html`, `app.js`, `styles.css`：網頁閱讀器應用程式檔案。
