此工作流會掃描 `章稿` 目錄，檢查正文中是否出現違禁詞。

### 執行步驟

1. 運行以下命令進行初步檢查：

// turbo
```bash
grep -rnE "(第[一二三四五六七八九十0-9]+卷|第[0-9]+章|Ch[0-9]+|Chapter [0-9]+|卷末|本章)" /Users/george/Library/CloudStorage/OneDrive-MSFT/0.專案/novel/novel_Ratforge/章稿 | grep -v "# 第" | grep -v ">"
```

*註：`grep -v "# 第"` 是為了過濾掉正常的章標題，`grep -v ">"` 是過濾掉設定或提示框。*

2. **人工審核**：grep 的結果可能包含合法的對話（如書中書的章節）或漏網之魚，請逐一核對。

3. **修正方式**：
   - 將「在第一卷中」改為「在最初的時候」。
   - 將「Ch45 那次」改為「在九號井的那次衝突中」。
   - 將「第五卷完」等刪除，或移至專門的卷末總結文件。
